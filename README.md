#First steps:

## Install
- `make build` - buduje obrazy
- `make` - uruchamia i loguje do kontenera##
- `composer install`
- `cp .env.example .env`
- `php artisan key:generate` 
- `php artisan migrate`

## Swagger
- `http://localhost/api/documentation`

## Command for topics import
- `php artisan import:topics-from-csv`
